## Setup

1. **Install Dependencies**:

   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

## Components

- **Canvas**: A dynamic background animation (`components/ui/canvas.tsx`).
- **Hero**: The main landing section (`components/ui/hero.tsx`).
- **Button**: Standard shadcn button (`components/ui/button.tsx`).

## Technologies

- **Framework**: Next.js
- **Styling**: Tailwind CSS
- **Design System**: shadcn/ui
- **Icons**: Lucide React

## Notes

- `dicons` was replaced with `lucide-react` for icon usage.
- Images rely on placeholders or standard styles.
